**Cars-Manager** is a full-stack Java application built with the Spring Boot framework. It provides a robust system for tracking vehicle lists, managing car-related tasks, and handling sales data.

**Key Features:**

**Car Inventory Management:** Add, update, and view vehicle details in real-time.

**Hybrid Database Storage:** Uses MySQL for structured data (users, security) and MongoDB for flexible car-related documents.

**Security:** Integrated with Spring Security for secure user authentication and access control.

**Modern Web UI:** Server-side rendering using Thymeleaf templates.

**AOP Logging:** Built-in Aspect-Oriented Programming for automated logging and monitoring.
